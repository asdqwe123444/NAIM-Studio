<li style="margin:4px 0 4px 16px;">앱 설치 없이도 데이터가 <b>영구 보존</b>됩니다</li>
<li style="margin:4px 0 4px 16px;">용량 제한이 거의 없어서 이미지·설정·프리셋 대량 저장도 가능합니다</li>
<li style="margin:4px 0 4px 16px;">브라우저 데이터(캐시·쿠키 등)를 전체 삭제하면 <b>같이 삭제됩니다</b></li>
<li style="margin:4px 0 4px 16px;">다른 기기로 옮길 때는 <b>프리셋 내보내기·세션 백업</b> 기능으로 JSON/ZIP으로 뽑아서 옮겨야 합니다</li>
